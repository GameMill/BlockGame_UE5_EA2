Control
- W = Forward
- S = Back
- A = Left
- D = Right
- Left mouse button = destroy
- Right mouse button = spawn / interact
- 1 = normal block (white)
- 2 = strong block (red)
- 3 = shrinkable block (green) # Interactable
- 4 = Light block (yellow) # Interactable
